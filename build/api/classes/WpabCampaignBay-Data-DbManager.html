app/app/Data/DbManager.php : 37

Database Manager Class.

This class is responsible for creating and maintaining the custom database tables required by the CampaignBay plugin. It uses the dbDelta function to ensure that tables are created and updated correctly upon plugin activation.

It is designed as a singleton to ensure a single instance is used throughout the activation process.
