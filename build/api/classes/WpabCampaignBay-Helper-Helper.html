app/app/Helper/Helper.php : 37

Helper Class.

This class provides a collection of static utility methods accessible globally within the plugin. It is designed as a central place for reusable logic, including fetching and filtering campaigns by type, calculating discount prices, generating promotional messages, and creating frontend HTML elements.
